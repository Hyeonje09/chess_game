# chess_game
3학년 2학기 컴퓨터 그래픽스 II 과목 기말 프로젝트, Chess Game입니다.

## 개발 기간
- 2022.11.06 ~ 2022.12.18

#### 멤버 구성
- 장현제 : OBJ 파일 업로드 및 체스 기물 정렬, 체스 경로에 따른 행마 구현, 기물을 잡았을 때의 체스 판 변화 구현
- 김창범 : 체스 맵, 카메라 시점 및 빛, 재질 표현, 규칙에 따른 기물의 경로 구현

#### 개발 환경
- 사용 언어 : C/C++
- 라이브러리 : OpenGL
